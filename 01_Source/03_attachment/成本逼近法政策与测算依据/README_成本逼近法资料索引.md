# 成本逼近法政策与测算依据索引

更新日期：2026-06-12

## 一、整理原则

- 本目录为复制汇总目录，原附件路径均保留不动。
- “通用政策标准”可用于建立按地区、地类和估价期日匹配的标准库。
- “项目调查参考”只用于参考正文结构和调查口径，不得自动套用于新项目。
- 政策文件能够证明收费依据，不等于一定能够直接给出当前项目的标准值。
- 每份报告开始前仍需冻结项目实际地类、面积、开发程度和调查证据。

## 三、下洞村秀鱼塘项目资料核查

已复制：

- `05_项目调查参考/下洞村秀鱼塘商服项目/下洞村秀鱼塘商服项目_成本逼近法报告.docx`
- `05_项目调查参考/下洞村秀鱼塘商服项目/下洞村秀鱼塘商服项目_测算表.xlsx`

可复用的调查口径：

- 报告给出道县城区“五通及场地平整”分项范围：
  - 通路：`14-16元/平方米`
  - 供水：`12-14元/平方米`
  - 排水：`14-16元/平方米`
  - 通电：`12-14元/平方米`
  - 通讯：`5-7元/平方米`
  - 场地平整：`13-15元/平方米`
  - 合计：`70-82元/平方米`
- 报告及测算表采用土地开发费`82元/平方米`。

使用限制：

- 测算表“成本逼近法”页的土地开发费为单元格`V5`直接填写的常数，没有公式或证据链接。
- 项目附件中未发现独立的询价记录、工程预算、结算资料或盖章调查表。
- 同一测算工作簿的其他“成本法”页混有衡阳、湖北及历史政策资料，不得整体导入系统。
- 因此该项目可作为调查表结构和道县历史取值参考，但`82元/平方米`不得作为全局自动标准。

## 四、仍需补充的资料

### A. 建议优先补充的通用文件

1. 道县现行被征地农民社会保障费用地单位缴纳标准正式文件，需明确金额或计算公式；当前省级文件不能证明`30元/平方米`。
2. 新增建设用地土地有偿使用费在2023-12-19之后继续执行、调整或替代的正式文件，用于确认2026年仍可采用道县十三等`16元/平方米`。
3. 水利建设基金现行延续、调整或废止文件，用于确认当前是否仍按新增建设用地有偿使用费的一定比例计算。
4. 现行耕地开垦费文件的清晰可检索原文或正式网页版，需能够核实文号、生效日期、地区等别和耕地类型标准。
5. 道县或永州市涉及专业菜地时适用的新菜地开发建设基金现行标准；普通项目不涉及时可不提供。

### B. 每份报告开始前单独收集的项目资料

1. 土地开发费调查资料：建议至少三组同区域、同开发程度的近期询价、预算或结算资料，并列明调查日期、项目位置、开发程度、分项费用、来源单位和证明附件。
2. 与估价期日对应的官方LPR发布资料，用于投资利息率。
3. 投资利润率调查依据，如同类土地开发项目收益调查、行业资料或估价师调查说明。
4. 土地增值收益率调查依据。出让收支管理办法只能作为辅助政策，不能单独证明具体收益率。
5. 项目实际地类和面积组成、农用地转用和征收批准资料，用于判断耕地占用税、开垦费、森林植被恢复费等是否适用。
6. 青苗、地上附着物和建筑物现场清点表、规格、数量、照片及补偿确认资料。
7. 占用林地项目另需林地类型、公益林属性、是否位于城市规划区及项目性质证明。

## 五、建议下一步

1. 先把可直接计算的区片价、区片范围、耕地占用税、森林植被恢复费接成版本化标准库。
2. 人工复核并结构化永州市补偿办法及耕地开垦费扫描表。
3. 将土地开发费改成“项目调查包”：用户录入或引用三组调查资料，系统展示区间并建议取值，但不自动套用其他报告的`82元/平方米`。
4. 缺少现行政策或项目证据时显示“待校核”，不继续使用没有来源链接的固定标准值。

## 六、2026-06-12 当前配置化结论

- 被征地农民社会保障费用地单位标准无公开统一标准，由市县根据 `湘人社规〔2023〕1号` 确定。系统保留用户填写入口，测试默认 `30元/平方米`，正式项目由用户确认。
- 新增建设用地有偿使用费暂沿用 `湘财综〔2018〕42号` 的道县十三等 `16元/平方米` 配置；后续若取得更新文件，再新增配置版本，不覆盖已冻结报告。
- 水利建设基金暂沿用现有政策及新增建设用地有偿使用费 `10%` 口径；已补充湖南省水利厅相关实施细则作为依据储备。
- 耕地开垦费已有可用原文，现按中等水田 `6.6万元/亩` 折算 `99.00元/平方米` 配置。
- 土地开发费不做全局自动标准，由用户根据项目调查资料填写；下洞村秀鱼塘项目只作为结构和取值参考。
- LPR 已有本地 PDF，并提供中国银行官网更新查询入口；查询结果由用户确认后应用。
- 投资利润率、土地增值收益率继续由用户填写或确认，测试期间可采用范本默认值。
- 当前已新增 `src/config/cost_approx_policy_rules.json`，可持续扩展地区规则、风险方案和区位模板；未知地区或过期政策保持待校核。
